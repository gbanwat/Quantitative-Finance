A quantitative back-testing framework implemented in Python for moving-average crossover strategies, leveraging pandas, NumPy, and finance libraries to perform signal generation, portfolio optimization, and performance evaluation using key risk-adjusted metrics such as Sharpe ratio, cumulative returns, and maximum drawdown.
